ABOUT.

STUDIO BEKKOではお客様の素敵な想いをカタチにして、より多くの人に広めるお手伝いが出来ればと考えております。

お客様と一緒に考え抜き、ご納得いただけるものを創らせていただきます。

SERVICE.

  • ヒアリング

    お客様が抱えている課題やご要望を細かくヒアリングし、最適な提案をさせていただきます。

  • デザイン作成

    ヒアリングの内容に基づいてデザインの作成を行います。印象に残り、使いやすいデザインを制作します。

  • コーディング

    デザインデータを基にコードを書いていきます。開発後も保守・運用がしやすいコードを心がけます。

  • サイト公開

    問題がなければサーバーにアップロードを行い、サイト公開となります。

SKILLS.
HTML

適切なタグを使用し、見やすいコードを書くことを心がけます。ここでウェブサイトの構造を作成します。

CSS

HTMLで作成した構造にスタイルを追加します。 画面幅が切り替わっても綺麗に表示されるようなコードを書いていきます。

Java Script

テキストや背景などに適切なアニメーションを付与し、見る人の印象に残るようなものを作ります。

LOGO

Adobe Illustratorを用いてアイコンやロゴの作成ができます。与えたい印象やご希望のイメージに沿ったものを再現します。
